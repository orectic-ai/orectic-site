## Hotfix Applied

**Commit `d9498db`:** Unicode escape sequences (`\u2013`, `\u2014`, `\u2019`) were rendering as literal text in JSX content. Replaced all with actual Unicode characters (–, —, '). Root cause: escape sequences work in JS string literals but render literally in JSX text nodes.
